# 编译原理实验资源仓库

## 简介

本仓库包含了编译原理课程的多个实验资源，涵盖了词法分析器、进制分析、语法分析器以及SLR语法分析器等实验内容。每个实验都提供了详细的实验报告和相应的实验代码，旨在帮助正在为实验报告发愁的同学快速上手并完成实验任务。

## 使用说明

1. **下载资源**：你可以直接下载整个仓库或单独下载某个实验的资源文件。
2. **修改与使用**：如果实验要求与本仓库提供的资源有所不同，你可以根据需要对实验报告或代码进行适当的修改。
